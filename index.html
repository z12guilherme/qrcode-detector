Como Funciona

Este aplicativo utiliza tecnologias modernas para ler códigos QR de forma segura e eficiente:

  • Processamento Local: Todo o processamento acontece no seu dispositivo, sem enviar dados para servidores externos
  • Detecção Inteligente: Identifica automaticamente o tipo de QR code (URL, email, telefone, Pix, WhatsApp, etc.)
  • Compatibilidade: Funciona com qualquer código QR padrão, não se limitando apenas ao Pix
  • Offline: Após o carregamento inicial, funciona sem conexão com internet

Privacidade e LGPD

Respeitamos sua privacidade e estamos em conformidade com a Lei Geral de Proteção de Dados (LGPD):

  • Não armazenamos dados: Nenhum código QR, imagem ou informação pessoal é armazenado em nossos servidores
  • Processamento local: Todo o processamento acontece exclusivamente no seu dispositivo
  • Sem rastreamento: Não utilizamos cookies, analytics ou qualquer forma de rastreamento
  • Código aberto: O código fonte está disponível para verificação de transparência
